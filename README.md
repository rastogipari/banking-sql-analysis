#  Banking Transaction Analysis (PostgreSQL)

## Project Overview
This Project analyzes banking customer data, accounts, transactions, loans using PostgreSQL

## Database Tables
- Customers
- Accounts
- Transactions
- Loans

 ## Key Analysis Performed
  - Total Bank Balance
  - Total Deposits vs Withdrawal
  - Richest Customer Identification
  - Monthly Transaction Trend (DATE_TRUNC)
  - Customer Ranking using Window Function (RANK)
  - High Value Transcation Detection (CASE WHEN)

 ## SQL Concepts Used
 - JOIN
 - GROUP BY
 - ORDER BY
 - Aggregate Functions
 - DATE_TRUNC
 - Window Functions
 - CASE statements

 ## Tools Used
 - PostgreSQL
 - pgAdmin 4
 - GitHub
